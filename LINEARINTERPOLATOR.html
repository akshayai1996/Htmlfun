Root Solver Information

The custom-equation root solver uses Regula Falsi (linear interpolation) with a fixed practical accuracy of 10^-15. The iteration limit is fixed at 100.
This keeps the interface simple and avoids settings that do not improve results in JavaScript floating-point arithmetic.

How to Use This Calculator (First-Time Users)

  1. Enter your equation in the form of 3^x+3^-x-5, x^3-5x+2, sin(x), log(x), or similar. log means base 10, and ln means natural log.
  2. To calculate Y from X, enter a value in Evaluate y at x. The calculator will immediately show the Y value.
  3. To solve X for a known Y, leave the X field blank and enter the target value in Find x for y.
  4. For reverse solving, give a sensible Range start and Range end that actually contain the answer.
  5. Number of Intervals controls how finely the range is scanned before root solving begins.
  6. Tolerance controls accuracy. Smaller tolerance means a stricter answer, but the solver may need more work.
  7. The solver first scans for a sign change, then uses linear interpolation / Regula Falsi to narrow the answer.
  8. If there are multiple roots in the selected range, all detected roots will be shown.
  9. The table below the result shows the sampled x and y values for the chosen range.
  10. If the answer is not found, widen the range or increase the number of intervals.
Forward Interpolation:
y = y1 + ((x - x1)(y2 - y1)) / (x2 - x1)

Reverse Interpolation:
x = x1 + ((y - y1)(x2 - x1)) / (y2 - y1)
